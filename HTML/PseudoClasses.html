Pseudo-classes

Pseudo-classes são recursos que o CSS disponibiliza para selecionar elementos específicos na tela ou com base em suas características sem a necessidade de criarmos classes próprias para eles. É como se o browser já colocasse neles algumas classes especiais que estariam disponíveis para nós. Vejamos algumas:


e:first-child = pega o elemento (e) que seja primeiro filho de seu elemento-pai.
e:last-child = pega o elemento (e) que seja último filho de seu elemento-pai
e:first-of-type = pega o elemento (e) que seja primeiro filho levando em conta seu tipo
e:last-of-type = pega o elemento (e) que seja último filho levando em conta seu tipo
e:hover = ativa a estilização aplicada ao elemento (e) quando o mouse está sobre ele
e:nth-child(n) = serve como um contador para selecionar os elementos (e)

Para uma referência completa das pseudo-classes disponíveis visite W3Schools
